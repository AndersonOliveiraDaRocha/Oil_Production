# Gap_Minder

Oobjetivo desse repositório é armazenar os arquivos em R oriundos 
das aulas do curso da UDACITY.
Esse repositório está armazenando uma análise de dados (EDA) que
fiz em uma das bases de dados disponibilizada pela GAPMINDER, a base
de dados de produção de Petróleo ao longo dos anos.
Como trabalho no segmento de óleo e gás achei essa base ideal ao 
mundo que vivo.
